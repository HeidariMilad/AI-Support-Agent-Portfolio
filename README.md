# AI Support Agent Portfolio 🤖

A modern, interactive portfolio showcasing an AI-powered customer support system with comprehensive dashboard analytics, real-time metrics, and workflow visualization.

## 🚀 Features

### 📊 Admin Dashboard
- **Real-time KPI Metrics**: 70% auto-resolution rate, 1.2s response time, 30% escalation rate
- **Interactive Cards**: Animated hover effects with gradient backgrounds
- **Recent Conversations**: Live chat monitoring with status indicators
- **Knowledge Base Management**: Drag & drop file upload with progress tracking

### 🔄 Integration Flow
- **Visual Workflow**: Step-by-step integration process visualization
- **Multi-language Support**: Animated badge highlighting international capabilities
- **Dynamic Arrows**: Gradient-animated connection flows
- **Partner Integrations**: Salesforce, HubSpot, Zendesk, Slack, Teams

### 💬 Customer Chat Interface
- **Real-time Messaging**: Interactive chat simulation
- **AI Response Indicators**: Typing animations and status updates
- **Message Bubbles**: Modern chat UI with timestamps
- **Responsive Design**: Mobile-optimized chat experience

### 📈 Before/After Visualization
- **Performance Comparison**: Visual metrics improvement display
- **Cost Savings**: Clear ROI demonstration
- **Efficiency Gains**: Productivity enhancement showcase

## 🛠️ Technology Stack

- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS with custom animations
- **Icons**: Lucide React (modern icon library)
- **Animations**: Framer Motion for smooth transitions
- **Build Tool**: Create React App with modern tooling

## 🎨 Design Features

- **Modern UI**: Clean, professional design with gradient accents
- **Responsive Layout**: Mobile-first approach with breakpoint optimization
- **Custom Animations**: Subtle hover effects and loading states
- **Color Scheme**: Professional blue/purple gradient palette
- **Typography**: Clean, readable font hierarchy

## 🔧 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/HeidariMilad/AI-Support-Agent-Portfolio.git

# Navigate to project directory
cd AI-Support-Agent-Portfolio

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```
